# Task 1 - File Handling Utility

## Description
This Java program demonstrates:
- Writing data into a text file
- Reading file content
- Modifying/appending data into the file

## Technologies Used
- Java
- VS Code

## Files Included
- FileHandlingUtility.java
- sample.txt
- README.md

## How to Run

Compile:
```bash
javac FileHandlingUtility.java
```

Run:
```bash
java FileHandlingUtility
```

## Output
The program successfully creates, reads, and updates a text file.
