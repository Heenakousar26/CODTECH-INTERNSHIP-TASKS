# Task 2 - REST API Client

## Description
This Java application fetches weather data from the OpenWeatherMap API using HTTP requests.

## Features
- REST API integration
- HTTP GET request
- Displays live weather data
- JSON response handling

## Technologies Used
- Java
- REST API
- VS Code

## Files Included
- WeatherAPIClient.java
- README.md

## How to Run

Compile:
```bash
javac WeatherAPIClient.java
```

Run:
```bash
java WeatherAPIClient
```

## Output
The program fetches and displays live weather information.
