# Task 3 - Multithreaded Chat Application

## Description
This project is a Java client-server chat application using sockets and multithreading.

## Features
- Multiple clients
- Real-time messaging
- Socket programming
- Multithreading

## Technologies Used
- Java
- Socket Programming
- VS Code

## Files Included
- Server.java
- Client.java
- README.md

## How to Run

Compile:
```bash
javac Server.java
javac Client.java
```

Run Server:
```bash
java Server
```

Run Client:
```bash
java Client
```
