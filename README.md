# JAVA Internship Tasks

This repository contains all tasks completed as part of the Java Internship program.

## Internship Tasks Included

### Task 1 - File Handling Utility
A Java program demonstrating:
- File creation
- File writing
- File reading
- File modification/appending

### Task 2 - REST API Client
A Java application that:
- Connects to OpenWeatherMap REST API
- Sends HTTP requests
- Fetches live weather data
- Displays JSON responses

### Task 3 - Multithreaded Chat Application
A client-server chat application using:
- Java sockets
- Multithreading
- Real-time communication

### Task 4 - AI-Based Recommendation System
A Java recommendation system that:
- Accepts user preferences
- Recommends movies based on ratings
- Demonstrates recommendation logic
